## Analysis

The control condition (no identity token) establishes the baseline for each language. Key differences in baseline register across languages: English is conversational and structured, Swahili is formal and philosophical, Spanish is warm and social, Japanese is polite and structured with bold headers, Arabic is analytical and structured.

**Key finding:** The baseline register itself varies meaningfully by language, which affects how much room tokens have to shift the output. Languages with a looser default register (English) allow more token-driven differentiation. Languages with a stronger formal default (Japanese, Arabic) resist token perturbation.
